## 使用

```shell
# 下载依赖插件
npm install

# 建立缓存
npm run build:dll

# 开发环境
npm start

# 编译发布
npm run build
```

#### `Tips`

```shell
同一份示例代码 webpack 不同版本编译时间
# webpack4.x
webpack build in ~9600ms

# webpack3.x
webpack build in ~25000ms
```
